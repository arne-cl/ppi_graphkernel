General information

This page provides the software necessary for running the machine learning system for protein-protein interaction extraction described in Airola A, Pyysalo S, Björne J, Pahikkala T, Ginter F, Salakoski T, All-paths graph kernel for protein-protein interaction extraction with evaluation of cross-corpus learning, BMC Bioinformatics 2008, 9 (Suppl 11)(S2). We kindly ask that users of the software cite this paper. An earlier version of the paper was presented at BioNLP 2008.

NEW Errata available for the article here.

Five corpora were used for evaluating the system, the AIMed [1], BioInfer [2], HPRD50 [3], IEPA [4], and LLL [5]. Derived versions of these corpora containing parses created with the Charniak-Lease parser [6] and transformed to collapsed Stanford format [7] are made available on this page. These data files are based on the conversions available here.

Please note that these data files are further processed from the converted files and do not represent the only possible "interpretation" of the converted files: for example, these files do not include candidate interacting protein pairs corresponding to the self-interactions that appear in the AImed corpus.
